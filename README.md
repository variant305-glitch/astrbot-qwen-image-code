# astrbot-qwen-image-code

AstrBot 插件，通过上传代码或日志截图让LLM来分析错误或内容识别
因为QQ发送消息的限制，所以只能上传图片
目前没法批量上传，只能上传一张图片，
后续可能会更新通过上传图片的base64编码
闲着没事自己玩玩
测试
# 命令
    /帮我看看（默认是代码或日志服务）
     默认是查看查看图片中代码或者日志的错误并给出建议

    /查看看看 [你的需求]
     这个后面跟上需求之后，会优先处理你提出的问题
         比如：/帮我看看 这是什么
               /帮我看看 图中的人在干什么
